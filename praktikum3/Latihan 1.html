Apa itu Bootstrap CDN?

CDN (Content Delivery Network) adalah cara tercepat untuk menambahkan Bootstrap ke website tanpa perlu download file.

Cara Menambahkan Bootstrap CDN

1. Tambahkan CSS Bootstrap di <head>

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">

2. Tambahkan JavaScript Bootstrap sebelum </body>

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>

3. Pastikan ada Meta Viewport

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Dengan Bootstrap

<!-- HTML dengan Bootstrap -->
<div class="container">
  <button class="btn btn-primary">Button</button>
  <h1 class="text-center">Centered Text</h1>
</div>

<!-- Tidak perlu CSS tambahan! -->
Keuntungan: Lebih cepat, konsisten, dan sudah responsive!

Template HTML Bootstrap Lengkap

<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Bootstrap Website</title>
  <!-- Bootstrap CSS -->
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
  <div class="container">
    <h1 class="text-center my-5">Hello Bootstrap!</h1>
    <button class="btn btn-primary">Click Me!</button>
  </div>

  <!-- Bootstrap JavaScript -->
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
